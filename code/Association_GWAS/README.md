# Association GWAS

This directory contains GRNTWAS association scripts.

Use the repository-level `README.md` for the maintained installation and usage instructions.

Main scripts:

- `Association_GRNTWAS.py`: association with LD reference covariance files when `--LD_pattern` is supplied, or without LD when it is omitted.
- `utils.py`: shared association I/O, tabix, LD, and SNP utilities.
